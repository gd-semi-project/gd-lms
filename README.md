# 🎓 GD-LMS (학사 관리 시스템)

## 🔗 GitHub Repository
https://github.com/gd-semi-project/gd-lms

---

## 📌 프로젝트 소개
GD-LMS는 대학 학사 관리를 위한 웹 기반 LMS(Learning Management System)입니다.  
관리자, 교수, 학생의 역할(Role)에 따라 각기 다른 기능을 제공하며,  
강의 개설부터 수강 신청, 출석 및 성적 관리까지 학사 운영에 필요한 전반적인 기능을 구현한 시스템입니다.

## ✨ 주요 기능

### 🛠 관리자
- 강의 개설 승인 / 반려 관리
- 학사 일정 관리
- 학생 / 교수 / 강의 정보 관리
- 통계 및 현황 조회

### 👨‍🏫 교수
- 강의 개설 신청
- 담당 강의 목록 조회
- 강의 상세 정보 조회
- 출석 관리 (회차별 출결 입력 및 수정)
- 성적 관리 (평가 항목 관리, 점수 입력 및 자동 계산)
- 수강생 명단 조회

### 👩‍🎓 학생
- 수강 신청 / 취소
- 개인 수강 내역 조회
- 시간표 자동 생성
- 출석 현황 조회
- 성적 조회

### 📚 공통 기능
- 사용자 인증 및 권한 관리 (로그인, 로그아웃, 비밀번호 암호화, Role 기반 접근 제어)
- 공지사항 및 Q&A 게시판
- 과제 및 자료 관리 (등록, 제출, 채점)
- 파일 업로드 / 다운로드
- 보안 처리 (XSS 필터링)

---

## 🧩 팀원별 담당 기능

### 👑 고희권 (팀장)
- 사용자 인증 및 권한 관리 모듈
- 공통 시스템 관리 모듈 (공통 예외 처리, 보안 설정)
- 파일 업로드 / 다운로드 모듈

### 📘 김현호
- 게시판 관리 모듈 (공지사항, Q&A, 글 작성/수정/삭제, 댓글)
- 과제 및 자료 관리 모듈
- 보안 처리 (XSS 방지 필터)

### 🗂 백시현
- 강의 개설 상태 관리 모듈 (요청 승인/반려)
- 수강 상태 실시간 통계 관리
- 학과 관리 모듈
- 학사 일정 관리

### 👩‍💻 신정근
- 수강 신청 관리 모듈
- 시간표 자동 생성 기능
- 신청 기간, 중복, 정원, 시간표 충돌 검증
- 출석 / 성적 조회 기능 (학생)

### 🎓최지윤
- 강의 개설 신청 및 강의 상세 관리 기능
- 출석 관리 모듈 (출석 상태 관리, 회차별 출결, 출석 통계, 출결 수정)
- 성적 관리 모듈 (평가 항목 관리, 점수 입력/수정, 성적 자동 계산, 성적 조회)
- 수강생 명단 조회 기능
- 출석 / 성적 관련 DB 테이블 설계 및 MVC 구조 구현
